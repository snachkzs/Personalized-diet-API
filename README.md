
### Setup

1. Clone repository:
```bash
git clone https://github.com/snachkzs/Personalized-diet-API.git
cd Personalized-diet-API
```

2. Install dependencies:
```bash
pip install fastapi uvicorn pydantic python-jose passlib bcrypt python-multipart
```

3. Run server:
```bash
python3 main.py
```

Server akan berjalan di: **http://localhost:8001**

## API Documentation

Setelah server berjalan, akses dokumentasi interaktif di:

- **Swagger UI**: [http://localhost:8001/docs](http://localhost:8001/docs)
- **ReDoc**: [http://localhost:8001/redoc](http://localhost:8001/redoc)
